## Summary
The checkout form does not validate whether the entered credit card expiry date is in the past.

## Steps to Reproduce
1. Add item to cart
2. Proceed to Checkout
3. Enter delivery address
4. Enter card number with expired date: `01/22`
5. Click Place Order

## Expected Result
Error: *"Your card has expired. Please use a valid card."*

## Actual Result
Order placed successfully despite expired card details.

## Impact
Payment processing failure at backend. Poor user experience. Potential security risk.
