## Summary
No error message is displayed when user enters an incorrect password.

## Steps to Reproduce
1. Open the login page
2. Enter valid email: `test@gmail.com`
3. Enter wrong password: `wrongpass`
4. Click Login

## Expected Result
Error message: *"Invalid email or password. Please try again."*

## Actual Result
Page reloads silently with no feedback to the user.

## Impact
User has no idea login failed. Could lead to confusion and security issues.
