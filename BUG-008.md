## Summary
After placing an order, the order confirmation page shows an incorrect or empty total amount.

## Steps to Reproduce
1. Add item to cart (e.g., ₹999)
2. Leave address field empty
3. Enter valid card details
4. Click Place Order

## Expected Result
Validation error OR correct total shown on confirmation page.

## Actual Result
Confirmation page loads with empty address and incorrect order total displayed.

## Impact
Customer confusion. Order records will have incorrect data in database.
