## Summary
Search returns all products on empty search, and shows unrelated results for gibberish input.

## Steps to Reproduce
1. Leave search bar empty → Press Enter
2. OR type `xyzabc123!!!` → Press Enter

## Expected Result
- Empty: Show "Please enter a search term"
- Gibberish: Show "No products found"

## Actual Result
- Empty: All products displayed
- Gibberish: Random products shown
