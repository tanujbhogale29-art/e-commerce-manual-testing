# 🐛 BUG-006 — Order Placed with Empty Cart

| Field | Details |
|-------|---------|
| **Bug ID** | BUG-006 |
| **Module** | Checkout |
| **Reported By** | Tanuj Bhogale |
| **Date** | March 2026 |
| **Severity** | Critical |
| **Priority** | High |
| **Status** | Fixed ✅ |

## Summary
A user can place an order with an empty cart by directly navigating to the checkout URL. Order is created with ₹0 total.

## Steps to Reproduce
1. Empty the cart (remove all items)
2. Navigate to `/checkout` directly via browser URL
3. Fill in address and payment details
4. Click Place Order

## Expected Result
Redirect to cart page with message: *"Your cart is empty. Add items to continue."*

## Actual Result
Order placed successfully with ₹0 total. Order confirmation shown.

## Impact
CRITICAL — Can be exploited to generate fake orders. Breaks order management system.

## Regression Result
After fix — TC15 re-executed → ✅ Pass
